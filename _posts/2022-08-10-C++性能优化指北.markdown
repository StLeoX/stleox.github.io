### 理解指令级别优化

    需要知道高级语言对应的运算指令、分支指令、访存指令等范畴。

    有如下的场景：

```cpp
int p = 0;
for(auto x : xs) {
  // a1
  if (p == 0) p = x;
  // a2
  p = x;
  foo(p);
}
```

    变量 p 初始化为 0 特殊值，程序逻辑决定 xs 中全体元素相等，**for 循环无法避免（但存在循环不变量）**。那么 a1，a2 哪种写法更高效？
    a1 开销：n 次分支；a2 开销：n 次访存。结论：给 a1 添加 likely。
